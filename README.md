# Лабораторные работы по курсу "Вычислительные алгоритмы" МГТУ ИУ7. 4 семестр, весна 2024.

## Преподаватели

- Градов В.М.
- Мальцева Д.Ю.

## Цель курса

Цель курса - научиться применять основные численные методы и понять принципы работы алгоритмов приближения и интегрирования. В рамках курса ставится задача освоить реализацию различных способов численного вычисления интегралов, построение аппроксимаций функций с использованием многочленов Ньютона, сплайнов и других методов, а также анализ точности, устойчивости и применимости выбранных подходов.
